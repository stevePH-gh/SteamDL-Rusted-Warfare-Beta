# Rusted Warfare SteamDL Beta

## A portable mod downloader
I created this project only for downloading Rusted Warfare mods from **SteamWorkshop**.  
I do not plan to add more games that this downloader would support.  

## Requirements:
- Windows 10
- Windows 11
- .NET Framework 4.8
- steamcmd

## For Mobile Version
**NOTES:** I have no plans to develop a mobile app version *(yet)*

## Initial Setup:
This program needs SteamCMD to work.  

**1**. Download steamcmd.zip  
**2**. Create a "steamcmd" folder on your desktop  
**3**. Extract "steamcmd.exe"  
**4**. Place "steamcmd.exe" inside "steamcmd" folder.  
**5**. Download mod files...
